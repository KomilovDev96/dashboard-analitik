Расчёт «Заказано»:

A = sum(УникальнаяЗаказПоставщику_Товары.Количество) — с дедупликацией по uuid + Номенклатура_Key, только где Отменено = false
B = sum(purchases.quantity) — только строки где purchase_order + product_id совпадает с uuid + Номенклатура_Key
Заказано = A − B
В таблице МРП теперь 4 колонки:

🟢 Остаток — текущий остаток на складе
🟠 В пути — закуплено, но ещё не поступило
🔵 Заказано — заказано у поставщика, ещё не в закупках
4 карточки в шапке: Всего записей / Остаток / В пути / Заказано
